# Get Started with MongoDB Atlas

MongoDB Atlas is a developer data platform, integrating a multi-cloud database with a diverse set of data services. Atlas simplifies database deployment and management while enabling modern data-driven application development.

## Register a new Atlas account

Signing up for MongoDB Atlas is completely free and doesn't require a credit card. Use the link below to create your account:

- [Try MongoDB Atlas!](https://mongodb.com/try?utm_campaign=devrel&utm_source=workshop&utm_medium=cta&utm_content=google_cloud_workshop&utm_term=stanimira.vlaeva)

## Deploy a free cluster

It's time to deploy your first forever-free database in MongoDB Atlas! You’ll do that using the Atlas GUI, so before starting, make sure you’re logged into your [account](https://cloud.mongodb.com?utm_campaign=devrel&utm_source=workshop&utm_medium=cta&utm_content=google_cloud_workshop&utm_term=stanimira.vlaeva).

1. Click on the **Build a Database** button.

    <img
        alt="'Create a database' section in MongoDB Atlas highlighting the 'Build a database' button" 
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/create-a-database.png')}
        width="500"
    />

1. Select the free **M0** tier.

    <img
        alt="Cluster type selection page highlighting the shared cluster option"
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/select-cluster-type.png')}
    />

1. Select a cloud provider and the region closest to your physical location.

    <img
        alt="'Cloud Provider & Region' section highlighting Google Cloud and the region named 'Belgium (europe-west1)'"
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/select-cloud-provider-region.png')}
    />

1. Expand the **Cluster Name** section and rename your cluster to **Workshop-Bakery**.

    <img
        alt="The expanded 'Cluster Name' section with an input field filled in with the text 'Workshop-Bakery'"
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/cluster-name.png')}
    />

1. Click **Create Cluster** to deploy your cluster.

    <img
        alt="Cluster creation page highlighting the 'Create Cluster' button"
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/create-cluster-button.png')}
    />

1. To navigate back to your deployments, select **Database** from the left-hand menu.

    <img
        alt="Security quickstart page highlighting the 'Database' tab in the left-hand menu"
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/security-quickstart-highlighted-database-button.png')}
    />

1. Creating the cluster may take a couple of minutes. Once the deployment is complete, you'll see your first cluster in the list!

    <img
        alt="The 'Database Deployments' page with a cluster named 'Workshop-Bakery'"
        src={useBaseUrl('/img/chapter-1-mongodb-atlas/database-deployments.png')}
    />
