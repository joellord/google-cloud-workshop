# Set up Google Cloud


:::tip
This section requires a Google Cloud account. Thankfully, the folks at Google have provided us with a [QuikLab](https://mdb.link/gc-lab-2) that you can use.
Get your [token](https://docs.google.com/spreadsheets/d/1_dzseizy-R5QGcfbnTM1l_V5Fkr86hPcN_zbwG09P14/edit?usp=sharing&resourcekey=0-rcTp81JC5v1xnxyURdy0bg) for the lab and follow the instructions below.
:::


In this section, we'll be using the Google Cloud Natural Language API to analyze the sentiment of the comments left on the cakes in the web app. That's why you need to set up a Google Cloud account and project.

1. Sign in to [Cloud Console](https://console.cloud.google.com/getting-started) and create a new project.

    <img
        alt="Select project dropdown in Google Cloud"
        src={useBaseUrl('/img/chapter-3-nlp/select-project.png')}
        width="500"
    />

    <img
        alt="Dialog box in Google Cloud highlighting the 'New Project' button"
        src={useBaseUrl('/img/chapter-3-nlp/new-project.png')}
        width="500"
    />

1. Remember your project ID. The rest of the tutorial will refer to it as `PROJECT_ID`.

    <img
        alt="New Project dialog in highlighting the Project ID text"
        src={useBaseUrl('/img/chapter-3-nlp/project-id.png')}
        width="300"
    />

1. [Enable billing](https://console.cloud.google.com/billing) to use Google Cloud resources. If you’re a new user, you’re eligible for the [$300 Free Trial](https://cloud.google.com/free) program.
